Chapter 10 · 5부

조별 실습과 마무리

수강생이 자기 업무 문제를 하나 고르고, Copilot으로 결과물을 만들어 발표하는 마지막 운영안입니다.

목차이전 장배포

조별 실습 과제

각 조는 실제 업무에서 반복되는 문제 하나를 고르고, Copilot으로 초안 또는 화면을 만듭니다. 결과물은 완성품일 필요가 없지만, 다른 사람이 보고 피드백할 수 있어야 합니다.

선택 A
메일/보고서 자동화
긴 메일이나 문서를 요약하고, 보고용 초안을 만든다.
선택 B
데이터 분석
CSV/Excel 샘플 데이터를 분석해 우선순위를 제안한다.
선택 C
MVP 화면
업무 아이디어를 HTML 대시보드나 발표 페이지로 만든다.
선택 D
에이전트 설계
반복 질의응답을 맡길 업무형 에이전트 설계안을 만든다.

60분 운영 스크립트

0~5분: 조별로 문제를 하나 정합니다. 너무 큰 문제는 피하고, 오늘 안에 초안을 만들 수 있는 문제로 좁힙니다.
5~15분: Copilot에게 첫 요청을 보냅니다. 목표, 자료, 출력 형식을 반드시 넣습니다.
15~35분: 결과를 검토하고 두 번째 요청으로 개선합니다.
35~45분: 발표할 결과물을 정리합니다. 화면, 요약문, 프롬프트 중 하나는 반드시 보여줍니다.
45~60분: 각 조가 3분 발표하고 2분 피드백을 받습니다.

평가 루브릭

항목좋은 결과의 기준
문제 정의누구의 어떤 반복 업무를 줄이는지 분명하다.
프롬프트 품질목표, 맥락, 자료, 출력 형식, 검토 기준이 들어 있다.
결과물말 설명만이 아니라 요약문, 표, 화면, 설계안 등 보이는 산출물이 있다.
검토보안, 사실성, 숫자, 적용 한계를 확인했다.

마무리 멘트

오늘 배운 Copilot 활용은 한 번에 완벽해지는 기술이 아닙니다.
하지만 내 업무에서 반복되는 한 가지를 골라 매주 조금씩 바꾸면, 한 달 뒤에는 일하는 방식이 달라집니다.
다음 주에 바로 해볼 작은 실험 하나를 정해보세요.
AI를 잘 쓰는 사람은 AI에게 모든 것을 맡기는 사람이 아니라, AI가 만든 초안을 더 좋은 업무 결과로 바꿀 줄 아는 사람입니다.
