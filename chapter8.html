MVP란 무엇인가

MVP는 완성품이 아니라 대화를 가능하게 하는 최소 화면입니다. “이런 기능을 만들면 좋겠다”를 말로 설명하는 대신, 버튼, 표, 흐름이 보이는 화면을 만들어 서로 같은 그림을 보게 합니다.

실습: CS 현황 대시보드 만들기

Copilot만 사용해서 하나의 HTML 파일로 CS 현황 대시보드 MVP를 만들어줘.
요구사항:
- 상단에는 오늘의 CS 요약 숫자 4개 카드
- 가운데에는 채널별 문의량 막대그래프 느낌의 영역
- 아래에는 우선 처리 이슈 목록
- 데이터는 더미 데이터로 넣기
- CSS와 JavaScript를 한 파일 안에 포함
- 발표자가 바로 설명할 수 있게 화면 문구는 한국어로 작성
빈 화면을 만들지 않습니다. 그래프 라이브러리를 못 쓰는 상황이면 CSS 막대, 표, 숫자 카드처럼 실제로 보이는 요소로 대체합니다.
