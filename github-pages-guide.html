GitHub Pages 배포 방법

Hermes님도 같은 방식으로 자기 버전을 배포할 수 있도록 정리한 절차입니다.

목차교재

방법 A. GitHub 웹에서 배포

  1. 새 저장소를 만듭니다.
  2. 이 폴더의 HTML/CSS/JS 파일을 업로드합니다.
  3. Settings → Pages로 이동합니다.
  4. Build and deployment에서 Deploy from a branch를 고릅니다.
  5. Branch는 main, folder는 root를 선택합니다.
  6. 저장 후 몇 분 기다리면 Pages URL이 생성됩니다.

방법 B. 터미널에서 배포

git init
git add .
git commit -m "Add Copilot AX instructor textbook"
gh repo create copilot-ax-instructor-textbook --public --source=. --remote=origin --push
gh api repos/:owner/copilot-ax-instructor-textbook/pages -X POST -f source.branch=main -f source.path=/
주의: 공개 저장소로 배포하면 누구나 볼 수 있습니다. 민감정보, 내부 정책 원문, 비공개 자료는 넣지 않습니다.
