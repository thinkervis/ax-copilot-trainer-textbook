제작 원칙

통짜 안내문이 아니라 강사용 학습 교재

이 배포판은 AX_basic의 초심자 A to Z 친절도와 AX_standard의 전체 범위를 함께 반영합니다. AX 개념, 프롬프트, 보안, Copilot 접속, Outlook, PDF 보고서, Excel 분석, MVP 제작, HTML 배포, 에이전트, 조별 실습까지 따라가며, 각 장은 개념 설명, 쉬운 비유, 실습, 강사용 멘트, 예상 질문, 검토 체크리스트를 포함합니다.

AX_standard 수업 맵

5부 흐름

1부 기초
AX 개념 이해와 Copilot 준비
AX를 도구 교육이 아니라 일하는 방식 전환으로 설명하고, Copilot 접속과 첫 실습까지 연결합니다.
2부 업무
Outlook, PDF, Excel 실무 자동화
메일 요약·작성, PDF 보고서 요약, CSV/Excel 데이터 분석을 수강생이 바로 따라 하게 만듭니다.
3부 제작
MVP, HTML 발표자료, GitHub Pages 배포
Copilot만 쓰는 환경에서도 업무 아이디어를 화면과 배포물로 보여주는 흐름을 다룹니다.
4부 확장
Cowork와 나만의 에이전트
에이전트의 역할, 지식 범위, 배포 전 점검을 강사가 안전하게 설명할 수 있게 구성합니다.
5부 실전
조별 실습과 Q&A 운영
교육 마지막 60~90분을 프로젝트형 실습으로 운영하는 스크립트와 평가 기준을 제공합니다.
